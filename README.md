# AKS Premium Cinematic Website

A premium, cinematic one-page website showcasing AKS for AI services with TitanGate aesthetic and ActiveTheory-style motion design.

## Features

- **Immersive Hero Section**: 3D holographic elements with GSAP-powered parallax scrolling
- **Interactive 3D Graphics**: Three.js-powered holographic core with floating structures
- **Smooth Animations**: Framer Motion and GSAP for cinematic transitions
- **Responsive Design**: Fully responsive across all devices
- **Modern Tech Stack**: React 18 + TypeScript + Vite

## Sections

- **Hero**: Immersive 3D holographic hero with parallax effects
- **About**: Company overview and mission
- **Agentic**: Showcase of agentic AI capabilities
- **Services**: AI automation services
- **Pricing**: Transparent pricing with ROI calculator
- **WhyUs**: Competitive advantages
- **Contact**: Multiple contact channels (Email, WhatsApp, Telegram)

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- GSAP (GreenSock Animation Platform)
- Three.js + React Three Fiber
- React Router
- Lucide React Icons

## Getting Started

### Install dependencies
```bash
npm install
```

### Run development server
```bash
npm run dev
```

### Build for production
```bash
npm run build
```

### Preview production build
```bash
npm run preview
```

## Development

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run lint` - Run ESLint
- `npm run typecheck` - Run TypeScript type checking
